# Sistema de Cálculo de IMC

Aplicación desarrollada en Java con arquitectura MVC y POO para calcular el Índice de Masa Corporal (IMC).

## Funcionalidades
- Registro de datos (nombre, edad, peso, estatura)
- Cálculo automático del IMC
- Clasificación del resultado (Bajo peso, Normal, Sobrepeso, Obesidad)
- Consulta de resultados con color según clasificación
- Información del desarrollador

## Tecnologías
- Java
- Swing (GUI)
- Patrón MVC
